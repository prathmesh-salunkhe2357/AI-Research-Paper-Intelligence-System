# 📚 AI Research Paper Intelligence System

An AI-powered Research Paper Intelligence System that helps users analyze, search, summarize, and understand research papers using advanced Natural Language Processing (NLP) and Machine Learning techniques.

---

## 🚀 Features

- 📄 Upload and process research papers in PDF format
- 🔍 Semantic Search using Sentence Transformers and FAISS
- 🤖 Retrieval-Augmented Question Answering (RAG)
- 📝 Automatic Research Paper Summarization using BART
- 🔑 Keyword Extraction using KeyBERT
- 🏷️ Custom Named Entity Recognition (NER) for research-domain entities
- 📊 Topic Clustering for organizing similar research papers
- 💾 Embedding Caching for faster retrieval
- 🌐 Interactive Gradio Web Interface

## 📂 Project Workflow

1. Upload one or more research papers.
2. Extract text from the uploaded PDFs.
3. Generate embeddings using Sentence Transformers.
4. Store embeddings in a FAISS vector database.
5. Perform semantic search based on user queries.
6. Generate concise summaries using BART.
7. Extract important keywords using KeyBERT.
8. Detect domain-specific entities using Custom NER.
9. Display results through the Gradio interface.
